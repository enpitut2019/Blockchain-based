# Blockchain-based
Trusted IoT Devices

## プロダクト名

IDマネージメントシステム（仮)

## エレベーターピッチ

どこに行っても自分の信頼性が保証されるID管理サービスの提供

各機関が自分でIDを管理する必要がなくなり、脆弱性を含む機器からのアクセスを防げる

サービスの利用者は１度アドレスを登録すれば、毎度毎度認証処理(メールアドレス送信 -> メール確認、開く)の手間がなくなる
